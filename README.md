# Video Controls

This project is a video player with custom controls for an future project of youtube clone.

## Installation

Install videoproject with npm

```bash
  cd videoproject
  npm install
```

## Functionalities

- Pause video button
- Fullscreen mode

## Stack

**Front-end:** React-native
